## Setup Instructions

1. **Clone the Repository**  
   Use the following command to clone the repo:
   ```bash
   git clone https://github.com/mohitkesarwani/multi-agent-sdlc-framework.git
   cd multi-agent-sdlc-framework
   ```

2. **Install Dependencies**  
   Make sure you have [Node.js](https://nodejs.org/) installed. Run:
   ```bash
   npm install
   ```

3. **Set Up Environment Variables**  
   Create a `.env` file in the root directory with the following variables:
   ```env
   VARIABLE_NAME=value
   ```
   Adjust the values according to your project requirements.
